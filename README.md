# klg-boilerplate-module

copy form egg-init

Boilerplate for klg module quickstart. 

klg npm 库模板

## Installation

**Don't `npm i` this directly**

```bash
$ npm i -g klg-init
$ klg-init --type module demo
$ cd demo
$ npm i
```

See [klg-init](https://www.npmjs.com/package/klg-init) for more detail.
